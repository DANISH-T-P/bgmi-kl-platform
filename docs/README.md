# BGMI-KL Website

## Play. Connect. Win.

BGMI-KL (BGMI Kerala) is a community-driven platform designed to unite BGMI players across Kerala through tournaments, networking, squad recruitment, community events, content creation, and competitive gaming opportunities.

This repository contains the official BGMI-KL website source code.

---

## Project Vision

To create Kerala's largest and most trusted BGMI community platform where players can connect, compete, grow, and build lasting gaming relationships.

---

## Core Objectives

* Community Building
* Tournament Hosting
* Squad Recruitment
* Player Recognition
* Content Creator Support
* Gaming Education
* Esports Ecosystem Development

---

## Technology Stack

Frontend:

* Next.js
* TypeScript
* Tailwind CSS
* Framer Motion

Backend:

* Supabase
